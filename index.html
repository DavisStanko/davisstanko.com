Website and Email Server

❮
❯

I have set up my Website and Email Server using a combination of reliable tools and services. To start, I purchased my domain from Porkbun and opted for a VPS from Vultr. I use Vultr to host my website, email, and other miscellaneous projects. For the website, I utilize Nginx as the web server and Certbot for managing SSL certificates. As for the email server, I employ a robust stack consisting of Postfix, Dovecot, Spamassassin, OpenDKIM, Certbot, and fail2ban. These components work together to provide efficient email handling, spam filtering, domain signing, and security measures to safeguard my email communications.

Tabliss
